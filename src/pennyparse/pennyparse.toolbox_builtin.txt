## pennyparse builtin tools

> The tools require its web server running with `pennyparse serve`

- ToolName: img_metadata_px
- Scope: planner
- Cost: very low
- Example:
    `pennyparse tool img_metadata_px --path /path/to/example.jpg"`
- Params: None


## pennyparse[full] builtin tools

> The tools belong to this section relys on `pip install pennyparse[full]` and require its web server running with `pennyparse serve`

- ToolName: pdf_metadata
- Scope: planner
- Cost: very low
- Example:
    `pennyparse tool pdf_metadata --path /path/to/example.pdf"`
- Params: None

---

- ToolName: pdf2txt
- Scope: parser
- Cost: very low
- Example:
    `pennyparse tool pdf2txt --path /path/to/example.pdf"`
- Params: None

---

- ToolName: pandoc2txt
- Scope: parser
- Cost: very low
- Example:
    `pennyparse tool pandoc2txt --path /path/to/example.docx"`
- Params:
    - path:str
